# Security Policy

## Supported Versions

The following versions of GoSearch are currently supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |


> Note: Only the latest minor versions within the active release branch receive security updates.

## Reporting a Vulnerability

If you discover a security vulnerability in GoSearch, please report it responsibly.

### How to Report

- Open a private issue **or**
- Contact me: arshtiwari12345@gmail.com

Please include:
- A clear description of the issue
- Steps to reproduce the vulnerability
- Potential impact assessment
- Suggested fix (if available)

### Response Timeline

- You will receive an acknowledgment within **48 hours**.
- I aim to provide an initial assessment within **2 business days**.
- If the vulnerability is confirmed, a patch will be released as soon as possible.

### What Happens Next

- If accepted, the issue will be fixed and a security patch release will be published.
- If declined, I will explain why it is not considered a security issue.

I appreciate responsible disclosure to help keep GoSearch secure for all users. thanks!
